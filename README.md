# E-Commerce Project Workspace

Welcome to the E-Commerce platform workspace. This workspace contains the source code for the premium storefront and landing page modules.

## Workspace Structure

*   **`sacrosanct/`**: The primary client-side web application built with React, TypeScript, Tailwind CSS v4, and Vite. It contains the premium editorial landing page, loading animations, interactive layouts, and product catalog showcases.

## Quick Start

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed (version 18+ recommended).

### Running the Project

1. Navigate to the `sacrosanct` directory:
   ```bash
   cd sacrosanct
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the local development server:
   ```bash
   npm run dev
   ```

4. Build the project for production:
   ```bash
   npm run build
   ```

For detailed information about the client-side codebase and architecture, see [sacrosanct/README.md](file:///e:/1.HOST/ecommerce/sacrosanct/README.md).
